\doxysubsection{Detailed Description}
\mbox{\hyperlink{classGenericNode}{Generic\+Node}} Test description for a git push as well!!! Every Generic node has the following IO. 


\begin{DoxyEnumerate}
\item Subscriber to broadcast messages
\item request socket for CNS lookups (to get a port and register with the CNS)
\item Publisher to send out messages
\item Subscriber to receive messages
\end{DoxyEnumerate}

Node identification
\begin{DoxyEnumerate}
\item Node type -\/ \mbox{[}kinect $\vert$ pcd $\vert$ saver $\vert$ UKF $\vert$ etc\mbox{]}
\item Node id (random id)
\item Node name -\/ \mbox{[}kinect\+\_\+0 $\vert$ pcd\+\_\+0 $\vert$ saver\+\_\+5 $\vert$ UKF\+\_\+3 $\vert$ etc\mbox{]} 
\end{DoxyEnumerate}
